Button

Button is standalone component to trigger an action or redirect user to other locations. It consists of a bounding box with certain border, some text content, and an optional icon. It has 3 states: hover, active, and focus

Description

The button class should be used on either an <a> or an <button> element. When an icon is used, the button-icon class should be used with either an <i> or and <svg> element. It should only used as an standalone component. If similar functaionlities are reqiured in the context of a larger component, please use the designated elements under the component (see menu, navbar):

  • <button>: the main container
    • <button-icon>: the icon attached to the button
    • text description
