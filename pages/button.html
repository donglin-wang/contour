Button

Button can be used on:

  • <a>
  • <button>
  • <input type="submit">
  • <input type="reset">

It may contain elements with the following classes:

  • button__section to organize and customize elements
  • button--symbol to annotate the symbol representing the effect of the button

As seen in the examples below, button__section can be nested to create complex layouts.
