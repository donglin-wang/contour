Typography

The typography class is used to reset typography of a given element. It resets the following properties of its target:

  • color
  • text-decoration
  • text-overflow
  • font-size
  • font-weight
  • line-height
  • margin-inline
  • margin-block

Instead of having a global or element specific reset, Contour only resets elements of your choosing. In addition, you can create specific variants to fit the need of a particular element or context.

The resets used for used for the typography class is also used as default font configuration for all components at the root level, though they are prefixed with their respective name.
